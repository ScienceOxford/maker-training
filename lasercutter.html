Using our Laser Cutter

These instructions show how to physically use the machine - it assumes you already have a file in DXF format, and have chosen an appropriate material to cut.

Opening your file

Save your file onto a USB stick, and plug this into the lasercutter's computer.

Do not remove the things plugged in on the left - they control the lasercutter.


Log into the laptop using the Administration user - there is no password.

Optional: save your file from the USB stick into the 'Laser job uploads' -> 'Your Name' folder.

Open the software - Lasercut 5.3 - by clicking the icon on the desktop.


Click 'File' -> 'Import' (or keyboard shortcut Ctrl+I), which should place your design onto the workspace.


If you get any error messages, or your file doesn't show up on the screen, close the Lasercut software and try again. If you continue to get errors, check your original file for problems.


Preparing the file for cutting

To cut neatly, you must unite the lines of your design. This tells the laser to cut a single long line, instead of lots of small ones. See the before and after pictures below, the circle changes from four cuts to one:


Click 'Tools' -> 'Unite lines', then accept the default tolerance of 0.01.


If parts of your design need to be cut differently (for example, part should be a shallow cut or an engraving instead of cutting all the way through), you need to change the colour of this part of the design.

Click on the part of the design that should change colour, then click on one of the colour options from the bar at the bottom.

This will cause multiple cut options to show on the right-hand-side:


You can use the Up and Down buttons to rearrange the order (the laser will action them in order).

You can untick the checkbox to stop one from running - this can be helpful when running test cuts, or if you want to do cuts individually.
